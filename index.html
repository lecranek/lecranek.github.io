Posted on 2020-01-08

修改 Next 主题的 Scheme

Next 默认的风格是 Muse。从现有的四个风格里，修改为 Gemini。
在此风格的博客首页里，每篇博客有明显的分割。菜单在左侧，方便访问时转到别的页面。

操作如下：

  1. 打开 themes\hexo-theme-next_config.yml

  2. 找到 Scheme Settings

  3. 去掉 Gemini 前面的 #

搭建过程中的一些小问题

  1. 在Git中使用 hexo init 初始文件夹时，在 INFO Install dependencies 时提示

    WARN Failed to install dependencies. Please run ‘npm install’ manually!

    参考csdn文章解决，方法：

    在 Git Bash 中按 Ctrl+C 取消下载，然后删除文件夹中所有文件，重新执行上面的 hexo init 命令

  2. 在Git Bash使用指令 ssh -T git@github.com 验证账户情况时提示

    Warning: Permanently added ‘github.com,某IP地址‘ (RSA) to the list of known hosts.

    猜测可能与本机有多个Github账号有关？
